Branddock Everywhere — Instellingen

Koppel de extensie aan je Branddock-workspace. Maak een API-key aan in Branddock onder Settings → API & Connectors.

De URL van je Branddock-installatie, zonder pad erachter.

De doelgroep-notitie (vrije tekst die met elke herschrijf-opdracht meegaat) stel je in via de popup van de extensie.
